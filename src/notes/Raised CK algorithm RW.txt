Raised CK algorithm RW 

Causes 
• Trauma/exercise/seizure 
• Drugs - Statin, antipsychotics, ETOH, MD/coke 
• Systemic - Myositis, thyroid, steroids/Cushing
• Hereditary myopathy- Dmd, glycogen storage 

Check CK if muscle pain/weakness along with U&E - if AKI and raised CK consider rhabdo & admit 

Check if on a statin 

If CK <500 (4x ULN) & on a statin - continue but if > than this stop statin, fluid ++ and repeat 1mo & if resolved rechallenge (only do this if was <10x ULN <1000) 

If not on a statin and <500 usual transient - recheck 1-2w, but if 4-10x ULN (>500-1000) usually secondary cause 
• Bloods - U&E, LFT, TFT, bone, Mg, vit D & urine dip, consider inflamm markers and Ref to rheum esp if >50
