Quick starting contraception RW

Off licence but can help prevent pregnancy, means not having to wait till next cycle, often good after giving emergency contraception 

If pregnancy cant be excluded (which it usually can’t) unless 
• no intercourse since LMP 
• already in d1-5 
• fully BF, Post partum 4w or ToP 7d
• no UPSI in last 3w with negative pregnancy test

= Can quick start COCP (not Dianette as can feminise a male fetus) POP or implant, depot only if others can’t be used as can’t remove it if pregnant subsequently, copper IUD only for EC

Generally 7d precautions for COCP/implant/depot Vs 2d for POP when started out of cycle 

If starting after EllaOne cannot quick start any hormonal contraception til 5d after as it can interfere with action of EllaOne for EC!

Can use bridging ST measure before using LARC

If actually pregnant then stop contraception immediately and it is somehow an IUS/D then USS r/o ectopic and refer for removal in first TM
