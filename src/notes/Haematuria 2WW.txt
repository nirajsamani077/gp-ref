Haematuria 2WW

2WW 

>45 + 
• unexplained visible haematuria without UTI
• persistent visible haematuria after UTI treated 

>60 + 
• unexplained non-visible haematuria (>1 episode, UTI excluded) + either 
• dysuria/raised WCC

Non urgent referral for >60 with recurrent UTI

If <40 refer renal, otherwise urology for cytosocpy

If <40 with non-visible haematuria, no need for referral if normal BP, no proteinuria, normal renal function (so check BP, urine MC+S, UACR, UE)

So generally worry about visible haematuria, screen first if non visible haematuria 

Causes 
• spurious - beetroot/rhubarb, rifampicin/doxorub
• transient - UTI/renal stone, exercise, sex, period 
• persistent - CA, BPH, prostatitis, renal (IgA, GN)
