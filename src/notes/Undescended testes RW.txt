Undescended testes RW

Usually picked up 6-8w exam 

Types 
• True = Testicle lies along Inguinal canal path 
• Rectractile = Can be brought down 
• Ascending = previously down now back up 
• Ectopic = outside of Inguinal canal 

Exam - if not in testicle put left hand deep Inguinal ring pressing down and use right hand to base of scrotum, most in Inguinal canal can be felt 

Rx 
1) UL  = review 3mo (usually resolves) if not ref urology routine 
2) BL palpable in Inguinal canal = ref routine 
3) BL impalpable or genital abnormal = ref urgent 
Any surgical correction should be done 6-18mo 

Comps = Ca, fertility
