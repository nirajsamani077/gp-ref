CMPA RW

Most common food allergy usually <1 (more common then lactose intolerance in older)

Find out 
• Symptoms - speed of onset, which foods/milk 
• Full feeding hx from birth & mothers diet if BF
• FHx food allergy and atopy
• Check weight length plot 

IgE mediated - rapid onset (mins)
• Angioedema, urticaria, vomiting, wheeze 

Non IgE mediated - gradual onset (2-72h)
• Reflux, diarrhoea/constipation, blood/mucus in stools, colic, atopic ezcema, cough/resp sx, FTT

If IgE mediated = REFER all allergy clinic, SPT/RAST
• FF - EHF formula, if persist swap to AA formula 
• BF - Exclude cows milk from mums diet (supp with 1g calcium 10mcg vit D per day) 
• Mixed - do both 
• Involve paed dietician 

If non IgE mediated = generally manage with GP
• As above exclusion diet 
• Refer dietician 
• If improved 2-4w exclusion can consider home challenge to reintroduce & if sx recur confirms Dx, if not consider alt Dx, if ongoing issues refer 
• If confirmed without allergy clinic continue exclusion til 9-12mo then try milk ladder 
• Any severe features FTT persistent sx = refer 

Hypoallergenic formulas  
• EHF - Aptamil pepti 1/Nutramigen LGG1 from birth, up to 2/LGG2 after 6mo, SMA Althera . Nutramigen contains lactose so dont use if GI Sx
• AAF - Nutramigen Puramino, SMA Alfamino, Neocate LCP
• Any milk with a “2” should only be used >6mo as follow on milk 
• Amount 
   - <6mo = 10 x 400g tin /month 
   - >6mo = 6-8 400g tin /month 

If ezcema alone management much the same for non-IgE mediated with topical steroids & derm 

Use GP Infant Feeding Network website and milk ladder
