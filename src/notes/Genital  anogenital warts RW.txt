Genital / anogenital warts RW 

Termed condylomata accuminata - sexually transmitted (direct contact) causes by non oncogenic HPV 6&11 (protected by Gardasil 9 vaccination at school age up to 25 or for MSM <45)

Small fleshy slightly pigmented protuberances although many don’t have visible lesions 

If found - examine for any cervical/vaginal lesions & offer STI screen via GUM/Umbrella 
• If internal vaginal or anal cryo best & antimitotics like Warticon/Aldara off licence/use with caution 
• If cervical consider referral for colp if uncertain 

Treatment - is cosmetic & not curative, no treatment is an option as 1/3 spontaneously resolve after 6mo but recurrence is common - sensible to notify current partner to test but no need for formal contact tracing. If choosing to Rx 
• Cryo (once weekly for 4w - if single/few warts) OR Podophyollotoxin (Warticon) 0.15% cream BD 3d a week then 4d break for 4w then review 
 - If >50% gone = continue cryo or Warticon
 - If <50% gone = switch imiquimod (Aldara) 5% cream applied same way as Warticon for max 4mo
 - If not gone by then refer GUM or excise if able 
• Cataphen 10% ointment (green tea leaf extract) could be used same day (3d/w 4d break) max 4mo

In pregnancy - can’t use antimitotics (teratogenic), cryo or excision safest options, don’t need to have CS unless obstructive 

Refer if IC (esp HIV), pregnant, child (as ?sexual contact - to paeds) or suspicious/uncertain lesions
