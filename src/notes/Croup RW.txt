Croup RW

Common, usually mild, 6mo-6y, viral infection (parainfluenza) causing seal-like barking cough & stridor, coryza and iWOB 
• DDx - epigottitis/bacterial tracheitis (toxic), FBS, anaphylaxis 
• Don’t need to examine, keep calm, check iWOB

Use Westley Croup score to assess but all children should get dex 0.15mg/kg STAT (or pred 1mg/kg if not available )
• Mild = no iWOB, only stridor when distressed 
• Moderate = iWOB, stridor at rest = admit 
• Severe = above + looks unwell, lethargic, RR>60= 999 admission keep calm (also admit if IC/chronic).
