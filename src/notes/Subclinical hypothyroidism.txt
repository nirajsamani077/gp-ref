Subclinical hypothyroidism 

Raised TSH, normal T4, no/minimal symptoms 

Risk of progression to hypothyroidism up to 5%, increased if anti-TPO antibodies present so check in all (if negative rpt TFT 12w, if positive may treat)

Main factors to consider starting treatment are
• TSH
• Age 
• obvious symptoms 
• presence of anti-TPO antibodies 

If TSH <10 (and >4 as ULN)
• if asymptomatic repeat TFT 3-6mo
• if symptomatic & <65 trial thyroxine 
• if older >80 probably best to monitor 

If TSH >10
• even if asymptomatic start thyroxine <70
• if older >80 probably best to monitor
