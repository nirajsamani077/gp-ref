Dental for docs RW

GP shouldn’t treat dental issues with ABx but can give treatment for pain and signpost to dentist - not medicolegally covered

3 common situations
• Dental abscess - examine, if red flags airway or periorbital cellulitis to ED, otherwise dentist 
• Tooth came out - needs to be reimplanted within 1h otherwise tooth dies, usually straight to max fax, first aid = clean tooth holding from crown and irrigate socket with saline, slowly push tooth back into socket using bite til meet resistance and ask pt to bite down onto gauze, amox 7d then maxfax 
• Bleeding post extraction - low grade oozing normal first day, if not then damp gauze apply pressure 5min rinse with saline reapply gauze, TXA mouthwash or damp tea bag (tannins = anticoag)
