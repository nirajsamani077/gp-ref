Abnormal LFT algorithm 

In history - ETOH, BMI, drugs, viral hepatitis RFs

Urgent admission/USS/2WW Gastro review if 
• Jaundice (BR>40)
• ALT >300
• Very low albumin or raised INR (synthetic)
• Suspected malignancy 
• Acute encephalopathy, ascites etc

If isolated raised BR with rest of LFT normal usually Gilberts (5% pop) - unconjugated BR, rarely haemolysis 
• Repeat with split BR, FBC +/- retics/LDH
• If normal with UCBR diagnose Gilbert’s with info 

Hepatic LFT with normal BR (ALT>ALP)
• Repeat 1mo with AST, GGT, FBC +/- HBV/HCV if RFs and advise ETOH reduction if >14 unit 
• If abnormal USS liver + extended liver screen with HBV/HCV (EBV, CMV), autoantibodies (AMA, ASMA, ANA), Igs, ferritin + TSAT, caeruloplasmin, A1 antitrypsin, HbA1C 
• If USS suggests fatty liver and rest of liver screen negative go down that algorithm, if any other abnormality refer to Gastro (urgent if suspected malignancy)
• If ALT >300 get urgent advice about above 
• If all negative including USS give lifestyle advice and repeat LFT in 1 year 

Cholestatic LFT with normal BR (ALP >ALT)
• If GGT also raised suggest liver so do USS and refer if abnormal 
• If GGT not raised consider bone, check bone profile and Vit D 

LFT normal in 25% with cirrhosis
• If harmful drinking (M >50 unit, F >35 unit) or AUDIT score >7 with normal LFT do an USS liver anyway and ELF test (separate gold top) with referral for fibroscan if >9.8 (high risk of advanced fibrosis)
