B12 deficiency algorithm RW

Symptoms /presentation 
• Sensory motor neuropathy, SACCD, ANS, gait, confusion/memory (ie neuro sx) - if any seek urgent advice 
• Subtle (loss of taste smell), glossitis, subfertility 
• Macrocytic megaloblastic anaemia, on metformin (1/10, consider testing if taking) and PPI 
• Folate and copper def (post GI surgery) mimic 

Diet - meat fish cheese eggs, so vegans def, some foods like cereals breads fortified 
Autoimmune - pernicious anaemia 
Malabsorption - Coeliac, GI surgery, pregnancy 

Always check folate - if deficient in both replace b12 first as otherwise risk of SACCD 

Scenarios 
1) Low B12 (<200) but NO SYMPTOMS
• If <150 check IF Abs and start B12 - Hydroxocobalamin 1mg IM 3x /week for 2 weeks loading then choice for pt depending if diet is a factor but generally IM every 3 months for life if clinical response (if not consider retest 1y), if IF Ab positive then lifelong IM every 3 months 
• If 150-200 repeat in 1 month (as may be false positive) - if persistent 150-200 check IF Ab and if positive down above line. But if negative IF Ab consider oral cyanocobalamin 50-150mcg/day and repeat in 3 months (if neuro sx develop urgent rv)
    - <150 = lifelong IM with loading as above 
    - 150-200 = 2nd line test (several choices homocysteine/methylmalonic acid, check with lab)
    - >200 (normal) = no further tests, can consider long term oral 150 if malabsorption (not licensed)
    - If clear dietary (vegan) - can have IM 6 monthly maintenance or oral lifelong (50-150 cyan, OTC)

2) Low B12 (<200) + SYMPTOMS (neuro urgent)
• Straight to checking IF Ab and starting treatment with IM loading and Maitenance as above as true deficiency much more likely (even if IF negative if clinical response consider 3 monthly IM lifelong )
• Any neuro involvement = urgent haem input as may be given alt day IM loading til improvement may need for longer than 2w and maintenance IM is 2 monthly (rather than 3 monthly)

3) Normal B12 + SYMPTOMS
• Check IF Ab and a second line test 
• Start B12 loading, if clinical response continue or either tests above positive then continue IM maintenance 
• If no response and both tests neg then unlikely to be deficient so no further tests or treatment 

Generally when on treatment don’t ever need to recheck B12, macrocytic anaemia should take 3 months to improve (retics response after a week, so if uncertain check retics after a week of loading and if suboptimal response consider other cause for anaemia eg IDA) and neuro sx 6 months to improve 
• Usually B12 well tolerated can cause itching/flushing/GI upset
