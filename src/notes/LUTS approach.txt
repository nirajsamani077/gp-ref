LUTS approach 

Voiding - hesitancy, stream, straining, incomplete emptying, terminal dribbling = BPH
• conservative (prudent fluid intake, reduce ETOH/caffeine) 
• Alpha blocker (tamsulosin) if moderate-severe IPSS
• 5-a reductase inhibitor (finasteride) if enlarged prostate & high risk of progression but affects PSA
• Both if both 

Storage - urgency, frequency, nocturia, incontinence = overactive bladder 
• conservative (prudent fluid intake, reduce ETOH/caffeine)
• bladder retraining 
• pelvic floor exercises 
• antimuscarinics if persists (oxybutynin/tolteridone IR or darifenacin), mirabegron if fails which is a beta 3 agonist & better tolerated 
• if nocturia consider furosemide/desmopressin

For all 
- Urine dip/MC+S - rule out haematuria/infection 
- PR
- PSA, counsel first 
- Urine frequency/volume chart 
- IPSS for impact, mild/mod/severe (>20)
